GIT flow
  • GIT clone

    - для того, чтобы "склонить" репозиторий

  • GIT pull

    - для того, чтобы получить изменения из репозитория

  • GIT checkout -b branchname

    - создать новую ветку и переключиться на неё

  • GIT add .

    - добавить изменения, которые сделали локально

  • GIT commit -m "commit text"

    - подписать их

  • GIT push origin branchname

    - запушить на git-сервер
